# OmsimSocial Core v2 Retention Blueprint

## Activation Definition
Activated by Day 2 when user:
- Creates 1 circle
- Invites or adds 3 people
- Posts 1 memory

## Day 0 (Signup)
- Demo Feed shown until first connection or first real memory
- QuickStart missions (0/3 -> 3/3)
- Nudge after onboarding: "Create your first Circle"

## Day 1-2
- Prompt to request Adoption Notes after posting a memory
- Suggested note chips for adopters (e.g., "What I felt", "What I learned")
- Adoption notes push always allowed (high-value)

## Day 3-7
- Circle prompts: 3 prompts in Week 1 (Mon/Wed/Sat)
- Replies encouraged from members
- Weekly recap on Day 7 (top moments + new adoptions)

## Day 8-30
- Vault habit nudges (1 gentle reminder per week)
- "On This Day" starts after Day 14
- Time Capsule nudge at Day 21
- Monthly recap on Day 30

## Notification Limits
- Max 3/week total
- Adoption-note push always allowed
- 1/day max for all other notifications

## Churn Detection
- Gentle push at Day 7 and Day 14 if no new memory or connection
- Only send if relevant (e.g., no circles or no vault activity)

## Success Metrics
- % activated by Day 2
- Adoption note received by Day 7
- Vault usage by Day 30 (adoptions + time capsules)
